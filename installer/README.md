# WinCC OA Installer

**Place the WinCC OA installation ZIP file here.**

## Required File

Download from: https://www.winccoa.com/downloads/

- **File name:** `WinCCOA-3.21.0-debian.x86_64.zip`
- **Version:** WinCC OA 3.21 for Debian Linux (x86_64)
- **Size:** ~1-2 GB

## Instructions

1. Download the ZIP file from the Siemens WinCC OA Portal
2. Place it in this directory: `installer/WinCCOA-3.21.0-debian.x86_64.zip`
3. Run `make build` or `./scripts/build.sh`
